# 🌑 LONG NIGHT

Браузерный arena-roguelike в духе Vampire Survivors. Ты — **охотник на нечисть** (магия и оружие на стороне Света); задача — пережить бесконечную ночь. Финальный босс — вампир. Один HTML-файл, без внешних библиотек: Canvas 2D + Web Audio API. Ставится на телефон как приложение (PWA) и работает офлайн.

**▶ Играть: https://emorozoff.github.io/long-night/**

![LONG NIGHT](art/menubg.jpg)

## Управление
- **Десктоп:** `WASD` / стрелки — движение, `ESC` — пауза. Оружие стреляет автоматически.
- **Телефон:** веди пальцем по экрану — герой идёт за пальцем.

## Что внутри
- **21 тип врагов** + **8 боссов** со своими механиками: Матерь роя, Людоед, Гнилой палач, Утопленница, Тёмный пастырь, Могильщик, Бледная дева, Мёртвый король (финал).
- **9 орудий** в тематике Света (арбалет, серебряные клинки, цепная молния, кадило, крест, святая вода, колокол, фонарь, меч веры) — лимит 4 за забег.
- **Эволюции** оружия и **слияния** (фьюзы двух орудий в одно мощное).
- **4 героя** со своими стартовыми предметами и спрайтами, **5 локаций**.
- **5 авторских музыкальных треков** (бесшовный луп, переключение по интенсивности/боссу).
- **Мета-прогрессия:** золото с подобранных монет → «Склеп» перманентных улучшений, бестиарий, достижения, экспорт/импорт сейва.

## Технологии
Вся игра — один файл `index.html` (~3500 строк): HTML + CSS + JavaScript, Canvas 2D, Web Audio API, DOM-оверлеи для интерфейса. Весь пиксель-арт генерируется в коде. PWA через `manifest.webmanifest` + `sw.js` (офлайн, установка на устройство). Приоритет — производительность: сотни врагов на телефоне.
