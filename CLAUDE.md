## 1. TL;DR

## 3. Рабочий процесс (КАК тестировать и деплоить) — критично
**Локальный тест:**
- HTTP-сервер: `python3 -m http.server 8773` в папке игры (фоном). Игра на `http://localhost:8773/game.html`.
- Headless Chrome с CDP на :9222 — запускать ТОЛЬКО с `--mute-audio` и УБИВАТЬ после тестов
  (`pkill -f "remote-debugging-port=9222"`). Иначе оставленный процесс играет музыку «из ниоткуда» (был такой баг — владелец искал источник звука по всему компу).
  Пример: `"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" --headless=new --mute-audio --remote-debugging-port=9222 --no-first-run --user-data-dir=/tmp/chrome-qa-profile --window-size=1280,800 about:blank &`
- Управление в тестах — через playwright MCP (`mcp__playwright__browser_*`): navigate/evaluate/screenshot/console_messages.
- **Debug-API `window.__game`:** `start() spawn(type,n,opts) boss(tier) warp(sec) god(bool) kill() levelup()
  pick(i) clearEnemies() count() fps() gold(n) meta()`. Плюс глобально доступны функции игры (refreshItems,
  damageEnemy, fireWeapons, poolAt, и т.д. — можно дёргать из evaluate для проверок).
- Синтаксис-чек без браузера: вырезать `<script>` и `node --check`:
  `sed -n '/<script>/,/<\/script>/p' game.html | sed '1d;$d' > /tmp/c.js && node --check /tmp/c.js`

**Деплой (GitHub Pages):**
- Скрипт-образец (использовался весь проект): клонировать репо в УНИКАЛЬНУЮ /tmp папку (`/tmp/ln_deploy_$$`),
  скопировать `game.html → index.html`, `ROADMAP.md`, `manifest.webmanifest`, `sw.js`, `icons/*`, `music/*`,
  закоммитить, `git push origin HEAD:main`. (см. историю — был `/tmp/deploy_ln2.sh`.)
- **GitHub-токен** лежит в macOS keychain: `printf "protocol=https\nhost=github.com\n" | git credential-osxkeychain get`
  → поле `password`. **gh CLI не установлен.** Аккаунт `emorozoff`.
- **ВАЖНО:** не запускать несколько клонов в ОДНУ папку параллельно (была гонка `index-pack`). Уникальная папка спасает.
- **CDN-лаг:** после пуша index.html обновляется ~1 мин, но новые БИНАРНИКИ (mp3/png) сначала отдают 404,
  затем 200 ещё через 1-2 мин — это нормально, подождать/перепроверить.
- **sw.js `CACHE`** (сейчас `'longnight-0.28'`) — ОБЯЗАТЕЛЬНО поднимать номер при каждом деплое, иначе у игроков
  залипают старые ассеты (документ грузится network-first, ассеты cache-first).

## 4. Архитектура game.html
Один `<script>`, секции (искать по `// ===== НАЗВАНИЕ =====`, не по номерам строк — они плывут):
`UTILS → AUDIO → ЛОКАЦИИ → META → INPUT → TOUCH → SPRITES → RUN STATE → UPGRADES → FX HELPERS →
COMBAT → GEMS/XP → WEAPONS → ENEMY BEHAVIOUR → BOSS → DIRECTOR → PLAYER/BULLETS/GEMS → UPDATE →
RENDER → MUSIC → MAIN LOOP → UI/SCREENS → WIRING → DEBUG API → BOOT`.

**Глобальное состояние:** `state` (строка: menu/heroes/pickLoc/shop/stats/best/settings/playing/paused/levelup/dead),
`run` (объект забега, пересоздаётся в `newRun()`), `meta` (сейв, localStorage + санитизация).

**Ключевые системы и функции:**
- **Спрайты:** `mkCanvas`, `px(rows,pal,s,outline)` — пиксель-арт из карты символов; `addOutline`, `whiteOf`
  (белая версия для вспышки урона). Враги — таблица `ENEMY_MAPS` (2 кадра ходьбы через `stepRows`/`shiftRow`).
  `SPR.*` — все спрайты (враги `SPR[type]`/`_a`/`_w`/`_wa`, снаряды `SPR.blade/scythe/pbolt/pbolt_g`,
  пикапы, иконки `SPR.icons`, тайлы `SPR.grounds`, декор `SPR.decors`).
- **Аудио:** объект `A` — синтез SFX + (отключённая в 0.15) процедурная музыка. `A.tone/noise`, компрессор.
  Файловая музыка — объект `MUSIC` (ленивая загрузка, кроссфейд по состоянию: menu/calm/intense/boss/tension/final).
  Тишина при master=0 — `A.ctx.suspend()` + разрыв цепи. Громкость синкается между вкладками (storage-событие).
- **Бой:** `fireWeapons(dt)` (все 8 орудий), `fireChain`, `damageEnemy(e,dmg,src,kx,ky)`, `damagePlayer(d,src)`,
  `killEnemy(e)` (дроп/комбо/делёж/осколки), `addBurst/addText` (партиклы/цифры), `hitStopAdd`, `flashGold`.
- **Враги:** `spawnEnemy(type,x,y,opts)`, поведение — switch по `e.type` в `updateEnemies(dt)`. Босс — `bossUpdate`/`spawnBoss`/`spawnFinalBoss` (`WIN_TIME=900`=15:00 → ВЕЧНЫЙ ВАМПИР → `winGame()`).
- **Спавн/сложность:** `director(dt)`, `diffNow()`, `pickType(t)` (веса по локации), `doWave`, элита/боссы по таймерам.
- **Левел-ап:** `addXP` → `openChoices` → `rollChoices`(`rollUpgrades`+эволюции EVOS)/`renderChoices`/`renderLvlActions`.
  `cardClick`→`pickChoice`/`banishCard`; `doReroll`/`toggleBanish`/`doSkip`. `applyUpgrade` (kind w/p/syn/evo).
- **Зоны:** `updateZones` (лужи swamp/crypt, разломы ash/inferno, кислотные лужи/горшки mortar, волны bell),
  `inPool/poolAt`. **Рендер:** `render()` → drawGround/drawZones/drawAura/drawGems/drawEnemies/drawPlayer/
  drawBlades/drawBullets/drawPots/drawFx/drawParts/drawTexts/drawMenuAmbient + торч-свет + виньетка + fps.
- **UI:** `setState(s)` + `SCREEN_MAP`, `refreshHud`, `refreshItems` (HUD: 4 слота оружия + пассивки),
  `buildHeroes/buildPickLoc/buildShop/buildStats/buildBestiary`, `icoURL/sprURL`, `recordRun`, `checkAchievements`.
- **Мета:** `defaultMeta/sanitizeMeta` (САНИТИЗАЦИЯ всех полей — не доверять localStorage), `metaBonus`, `saveMeta`.

## 5. Контент (на 0.28)
- **Оружие (8, лимит 4 за забег):** Арбалет(bolt), Серебряные клинки(orbit), Цепная молния(chain),
  Освящённая земля(aura — ещё и рассеивает вражеские снаряды), Метательный крест(scythe — бумеранг),
  Святая вода(mortar — лужа огня), Колокол(bell — волна), Фонарь(lantern — луч-конус).
- **Эволюции (EVOS):** оружие на макс + предмет-партнёр → супер-версия (Серебряный залп, Нимб, Гнев небес,
  Святилище, Распятие, Кара огнём, Благовест, Маяк). Гарантированно предлагаются золотой картой.
- **Пассивки (12) + синергии (7)** — массив `UPGRADES`. Имена в тематике Света.
- **Герои (4, `HEROES`):** Охотник(free), Паладин(300), Чернокнижник(450), Инквизитор(600) — свой стартовый
  предмет + моды статов. Выбор: В БОЙ → герой → локация.
- **Локации (5, `LOCS`):** Погост, Кровавая топь(лужи), Пепелище(разломы), Затопленный склеп(ледяные лужи),
  Преисподняя(частые разломы). Открываются по рекорду времени; в выборе ПОКАЗАНЫ ТОЛЬКО ОТКРЫТЫЕ.
- **Враги (6, `ETYPES`):** zombie/bat/slime/spitter/brute/warper + boss. ⚠️ В 0.29 довести до 20+ (см. §8).
- **Музыка (6 треков, `music/`):** menu.mp3, calm.ogg, intense.mp3, boss.mp3, tension.mp3, final.mp3
  (CC0 OpenGameArt + CC-BY Kevin MacLeod — атрибуция в настройках). Перекодированы в 128k, ~20 МБ суммарно.
- **Мета-прогрессия:** золото → «Склеп» (8 перм. улучшений `META_DEFS`), бестиарий, достижения (8),
  статистика (экран «Летопись»), экспорт/импорт сейва JSON, PWA-установка + офлайн.

## 7. Известные баги / подводные камни (hard-won)
- **Retina/DPR:** канвасу нужны И `cv.width=W*DPR`, И `cv.style.width=W+'px'` — иначе на Retina картинка
  уезжает в угол и мылится. Есть `watchDPR()` на смену dppx (перенос окна между мониторами).
- **TDZ:** `sanitizeMeta` НЕ должна ссылаться на `HEROES`/`EVOS` (объявлены позже) — вызывается при загрузке.
  Валидацию hero делать без списка (fallback в newRun/buildHeroes).
- **localStorage недоверенный:** всё гонять через `sanitizeMeta` (был баг с NaN-золотом, up:null).
- **Звук «из ниоткуда»:** см. §3 — headless без `--mute-audio` + не убитый процесс. При master=0 контекст suspend.
- **Деплой-гонка** в одну /tmp папку → уникальная папка. **CDN-лаг** бинарников (404→200). **sw CACHE** бампить.
- **Кристаллы:** при заваленном поле дроп XP сливается в ближайший кристалл, но с ПОТОЛКОМ ценности (≤100),
  иначе после босса — лавина из 20 уровней подряд (был баг).
- **update() прерывается** при смерти в кадре (state!=='playing' → return), иначе двойное золото/воскрешение.
- Текущих известных крашей нет; консоль чистая на 0.28. FPS держится при 250+ врагах (пиксель-спрайты дешевле).
