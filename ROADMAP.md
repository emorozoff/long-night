## Что это
Браузерный arena-roguelike в духе Vampire Survivors. **Один файл `game.html`** (HTML+CSS+JS,
без внешних библиотек): Canvas 2D + Web Audio API. Язык интерфейса — русский.

**Концепция (важно):** мы играем за **человека-охотника на нечисть**, который использует
**заклинания и оружие на стороне Света**. Главный босс — **вампир**. Раньше игра называлась
«Багровая Орда» и была «про вампиров»; сейчас переименована и переосмыслена.

- **Версия игры видна в меню.** Каждое изменение, даже мелкое = бамп версии (0.12 → 0.13 → …).
  Версия задаётся константой `VERSION` в game.html и показывается в меню + угловой плашкой.

> 1. **Босс роняет ОГРОМНУЮ кучу XP напрямую.** В `killEnemy` (game.html):

> 3. **Магнит зачищает ВСЁ поле разом.** Пикап магнита (`~2389`) ставит `mag=true` всем xp-кристаллам → `addXP` за серию кадров
>    прокручивает `while(run.xp>=run.xpNext){ run.level++; run.lvlQueue++; }` (`~1665`). `lvlQueue` копится до ~20 → 20 экранов выбора подряд.
>    Кривая `xpNeeded(l)=floor(6+4l+l^1.85)` (`~1163`): суммарно до 20-го ур. ≈ 2700 XP, до 24-го ≈ 4200 → **4350 XP даёт +~20 уровней**. ✔ сходится.

> ⚠️ Существующая защита `updateGems ~2357` (`if(state!=='playing') break;`) ограничивает лишь ЧИСЛО триггеров за кадр — `lvlQueue` всё равно копится до 20. Это и есть причина, почему «вроде чинили, а баг живёт».

### 2. Боссы и локации (план C) — большой арк

Прогрессия: открыть следующую локацию = убить финального босса текущей. Абсолютный финал — Мёртвый король.

**Новые механики арки:**
- **Препятствия:** редкие коллайдеры-кружки, детерминированно по миру (как декор, но «твёрдые»), тематичны

- **Тронный зал** при выходе Мёртвого короля **запечатывается** в зал-арену; до этого — обычное поле.

**Локации (5, переосмыслены под боссов):**
| # | Локация | Палитра | Препятствия | Финал |
|---|---|---|---|---|
| 1 | Затонувшая деревня | холодная болотно-зелёная с синью | сваи мостков, крыши, коряги | Утопленница |
| 2 | Выжженные пастбища | выцветшая охра, серость | кривые столбы ограды, черепа на кольях | Тёмный пастырь |
| 3 | Погост | бурая земля, мокрый камень, туман | кресты/плиты, кучи земли, лопаты | Могильщик |
| 4 | Заброшенная усадьба | холодный бело-голубой, серебро | мебель под простынями, колонны, зеркала | Бледная дева |
| 5 | Тронный зал | поблёкшее золото/багрянец, обугленный камень | разбитые колонны, трон, доспехи | Мёртвый король |

**Боссы (8, механики зафиксированы):**
1. **Матерь роя** (общая) — кайтит прочь; плодит рой-мошкару (**0 опыта**); только адды; фазы → гуще выводок.
2. **Людоед** (общая) — прёт на игрока; **пожирает монстров на пути → лечится/крепнет** (стак HP/урон/размер); ближний бой.
3. **Гнилой палач** (общая) — удары оставляют **ядовитые/гнилостные лужи**, area-denial.
4. **Утопленница** (лок-1) — **Прилив**: волны-телеграфы катятся по арене, dodge; после волны вода замедляет.
5. **Тёмный пастырь** (лок-2) — **Зов воронья**: стаи ворон волнами/линиями (живые снаряды) + наскоки.
6. **Могильщик** (лок-3) — **Поднятие мёртвых** (площадной спавн дохликов у игрока, **0 опыта**) + **разрытые могилы**-ловушки под игроком.
7. **Бледная дева** (лок-4) — **Зеркала**: двойники ВНЕШНЕ идентичны; тэлл по поведению — фантомы зеркалят движение игрока, настоящая охотится по-настоящему и белеет от попаданий.
8. **Мёртвый король** (лок-5, финал) — крутой мобильный дуэлянт, **бессмертен**. В центре зала трон, вокруг факелы; подходишь → факел зажигается, гаснет через N сек; горят ВСЕ одновременно → король смертен. При появлении короля все факелы тухнут (обучающий момент).

**Пилот:** первым делаем **Матерь роя** целиком (спрайт + механика), тестим, по ней решаем темп остальных.
Боссы рисуем в **текущем пиксель-пайплайне** (крупнее/детальнее), не внешними ассетами.

## Как тестировать (headless)
- Локальный сервер: `python3 -m http.server 8773` в папке игры.
- Headless Chrome с CDP: запускать **обязательно с `--mute-audio`** и убивать после тестов
  (`pkill -f "remote-debugging-port=9222"`). Иначе оставленный процесс играет музыку «из ниоткуда».
- Управление через playwright MCP (navigate/evaluate/screenshot).
- В игре есть debug-API `window.__game`: `start() spawn(type,n) boss(tier) warp(sec) god(b)
  kill() levelup() pick(i) count() fps() meta()`.
- DPR-баг был: на Retina нужен и `cv.width=W*DPR`, и `cv.style.width=W+'px'` (исправлено).

## Архитектура game.html (секции по порядку)
utils → audio (объект `A`, синтез SFX + процедурная музыка) → meta (defaultMeta/sanitizeMeta,
localStorage) → LOCS (локации) → input (+ touch-джойстик) → sprites (`buildSprites`,
пре-рендер в offscreen-канвасы) → run state (`newRun`) → UPGRADES (пул апгрейдов) → combat →
weapons (`fireWeapons`) → enemies → boss → director (спавн/сложность) → zones (лужи/разломы) →
update → render → ui/screens (`setState`, `SCREEN_MAP`) → wiring → debug API → boot.

Состояние забега — объект `run`, пересоздаётся в `newRun()`. Глобальный `state`:
menu/shop/settings/playing/paused/levelup/dead.

  кликнуть карту в режиме изгнания, предмет в run.banished, исключается из rollUpgrades),
  **Пропустить** (+12 HP, +◆6). Хоткеи R / B. `cardClick()` маршрутизирует pick/banish.
- Пикапы на поле (kind в run.gems): **магнит** (притягивает все кристаллы/золото),
  **бомба** (урон всем в радиусе 380), **монета** (золото). Падают: монета ~1.2% с обычных,
  магнит/бомба с элит (50/50) и с боссов. Магнитятся опыт/золото/сердца; сундук/магнит/бомба — поднять самому.

### 0.17 — эволюции оружия
- `EVOS` (массив): оружие на макс + предмет-партнёр на нужном ур. → гарантированно предлагается
  золотая карта «★ ЭВОЛЮЦИЯ» (приоритет в `rollUpgrades`). `run.evo[id]=true`, `applyUpgrade`
  обрабатывает kind 'evo'. Пары: Арбалет+Серебряные пули(crit3)→Серебряный залп (веер 7 болтов);
  Клинки+Песочные часы(haste3)→Нимб; Молния+Наручи(might3)→Гнев небес (удар по площади в точках);
  Аура+Кираса(armor3)→Святилище (шире+лечит); Крест+Библия(hunger3)→Распятие; Вода+Елей(regen3)→Кара огнём.
  Эффекты — ветки `run.evo.X` в fireWeapons/fireChain/updateZones. Болты эволюции золотые (`b.evo`).

### 0.19 — PWA (установка на телефон + офлайн)
- Файлы: `manifest.webmanifest`, `sw.js`, `icons/icon-{192,512,180}.png` (иконка — золотой крест +

  иначе старые ассеты залипнут у игроков. Деплой теперь копирует и эти файлы.
