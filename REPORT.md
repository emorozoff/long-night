# БАГРОВАЯ ОРДА — отчёт о разработке

Браузерный arena-roguelike в духе Vampire Survivors. Один файл `game.html` (~2000 строк), без внешних библиотек: Canvas 2D, Web Audio API, DOM-оверлеи для UI.

**Запуск:** открыть `game.html` в браузере (или `python3 -m http.server` для http-режима). localStorage используется как необязательный бэкап — при недоступности игра работает из памяти сессии с экспортом/импортом JSON.

| ≥6 типов врагов | 7: упырь (преследователь), нетопырь (рой, синусоидное движение, спавн пачками), слизень (делится дважды при смерти), плевун (держит дистанцию 330px, стрейфит, стреляет), громила (танк, иммунен к отбрасыванию, ярость при HP<30%), бес (телепорт к игроку с телеграфом + рывок), элита (×9 HP, нова из 9 снарядов, дропает сундук) |
| Левел-апы с паузой | Пауза мира, 3 случайные карты, выбор мышью или 1/2/3, очередь уровней |
| ≥12 апгрейдов + синергии | 17: 4 оружия (×5-6 уровней), 9 пассивок, 4 синергии с пререквизитами: «Стылая кровь» (сгустки замедляют, требует bolt≥3) → «Осколки боли» (замедленные взрываются при смерти, урон скейлится от HP жертвы); «Сверхпроводимость» (chain≥2 + aura≥1: +60% урона молнии в кольце, +2 цепи); «Кровавый вихрь» (orbit≥3 + вампиризм: клинки лечат и крутятся на 35% быстрее) |

| Босс каждые 3 мин | Тиры с растущим HP (1300·tier^1.5), 4 имени по циклу. 3 фазы по HP: веер прицельных снарядов → двойная спираль + рывок с телеграфом → кольцевые новы + призыв миньонов. Полоса HP сверху, стрелка-указатель за экраном, догоняющая скорость |
| Партиклы/скриншейк/цифры/трупы | Всё есть; декали крови затухают за 45с (кап 240), партиклы кап 600, цифры урона кап 90, криты крупнее и оранжевые |
| Web Audio | Синтез всех SFX (выстрел, попадание, смерть, подбор с растущим питчем-комбо, левел-ап, молния, рык босса…) + процедурная музыка: секвенсор с lookahead-планированием, прогрессия Am-F-G-E, слои (бас → бочка → хэты → снейр → арп → лид) включаются по интенсивности = f(время, число врагов, босс); BPM 100→146 |
| Мета-прогрессия | Золото за забег (убийства/время/боссы/элиты, ×«Жадность»), магазин «СКЛЕП»: 8 перманентных улучшений с растущими ценами (вкл. возрождение). Хранение: переменная сессии + экспорт/импорт JSON + localStorage-бэкап с санитизацией |
| Экран смерти | Время, уровень, убийства по типам, урон по каждому источнику, DPS, полученный урон, кристаллы, боссы/элиты/сундуки, прокачка забега |
| Меню/пауза/громкость | Главное меню (с рекордом и ambient: кровавая луна + нетопыри), пауза (ESC + автопауза при сворачивании вкладки), 3 слайдера громкости (master/music/sfx) |
| Баланс-кривая | HP врагов ×(1+0.55m+0.17m²), урон ×(1+0.22m), кап 40+32m (до 330). Минута 2 ≈ ×2.8 HP, минута 5 ≈ ×7 (на грани — проверено: стоя на месте смерть за ~10с), минута 10 ≈ ×23 + лавина спавна |

## Архитектурные решения

- **Один `<script>`, секционная структура** (utils / audio / meta / input / sprites / run / combat / weapons / enemies / boss / director / render / ui / debug). Состояние забега — один объект `run`, пересоздаваемый на рестарт: чистый рестарт без перезагрузки гарантирован конструктивно.
- **Spatial hash grid** (ячейка 72px) для коллизий пуль/клинков/ауры/расталкивания — 400+ врагов при 60 FPS.
- **Все спрайты пре-рендерятся** в offscreen-канвасы при загрузке (включая белые «вспышки урона» через composite source-in) — в кадре только drawImage.
- **Отложенное удаление** (`dead`-флаг + sweep в конце кадра) — безопасные взаимные убийства в цепочках (осколки → смерть → осколки).
- **Телепорт отставших врагов вперёд по курсу** (как в VS) — от орды нельзя убежать; боссы вместо телепорта получают догоняющую скорость (до ×3.4 на дистанции).
- **Debug API `window.__game`** (spawn/warp/god/kill/fps/levelup…) — игра тестируема headless-браузером.

## Ключевые балансные решения

- Снаряды, нацеленные на босса, получают +3 пробития — иначе толпа физически экранирует босса (найдено тестом: босс не получал урона в окружении 80 зомби).
- Молния приоритизирует босса в радиусе; один снаряд залпа гарантированно летит в босса.
- XP босса скейлится со временем (120·tier·(1+0.12m)) — убийство босса стоит риска.
- Урон «Осколков боли» = 10 + 15% maxHP жертвы — синергия не умирает к 5-й минуте.
- «Спешка» теперь ускоряет и ауру (период тика), и клинки (скорость вращения + пер-врагный КД) — нет «мёртвой карты».
- Ранний горб сложности срезан: рои нетопырей 2-4 (было 3-5), их вес снижен до 90с, первая волна в 0:55, орда 8-13.
- При переполненном поле кристаллы сливаются в ближайший (визуально и по перфомансу: нет «ковра» из 400 гемов).

## Циклы QA (headless Chrome + Playwright, скриншоты qa*.jpeg)

### Цикл 1 — смоук всех систем
Консоль чистая; 60 FPS при 424 врагах (16.5ms/кадр); рестарт без перезагрузки чистый; автопауза при сворачивании; магазин/экспорт/импорт работают. **10 найденных недостатков:** экран слишком тёмный (виньетка 0.95 душила картинку), босс тонул в толпе, нет указателя на босса, HUD не обновлялся в паузах, оверкилл в статистике урона (100014), игрок мелкий, пустые первые 20 секунд, нет отклика подбора, мелкие цифры урона, статичное меню. **Все исправлены.**

### Цикл 2 — визуал в бою
Найдено: ковёр из сотен кристаллов после зачисток, кровавая каша из 340 непрозрачных декалей, невидимые орбитальные клинки, бледная аура, слишком плотный оверлей меню, игрок тривиально убегает от спавна. **Исправлено:** слияние гемов, затухающие декали (45с), клинки крупнее с glow, аура ярче, спавн с упреждением по курсу (45%).

### Цикл 3 — игровые потоки
Проверены: сундук элиты («ДАР СКЛЕПА» — бесплатный выбор из 3 карт, +15 золота), убийство босса (баннер, +40 золота, скрытие полосы), возрождение «Второе сердце» (баннер, нова, 60% HP), второй тир босса, настройки из паузы, естественная смерть от орды на минуте 5 (~10с без движения — давление подтверждено), аудиодвижок (контекст running, секвенсор планирует вперёд). Исправлена вёрстка настроек.

### Цикл 4 — мультиагентное адверсариальное ревью + edge-тесты
Параллельно: прогон на 1024×640, лейт-гейм минуты 20+ (362 врага + 6 боссов, 60 FPS), и **workflow из 3 независимых ревьюеров (логика / баланс / edge-cases) с верификацией каждой находки отдельными агентами-скептиками: 24 находки → 14 подтверждено, 10 отклонено как ложные.** Найдена ключевая проблема геймфила: пустой экран при беге (отставшие враги не догоняют) → телепорт отставших + догоняющая скорость босса.

### Цикл 5 — исправление подтверждённых находок ревью
1. **[критика]** Подбор сундука в кадре смерти перетирал экран смерти, воскрешал забег и дублировал золото → ранний выход из `update()` при смене состояния.
2. Радиус поиска в grid был меньше суммы радиусов коллизии — снаряды «проходили сквозь» край босса (r=42 > поиск 32) → запас +44-50px во всех запросах.
3. Глобальный preventDefault ломал пробел/стрелки в textarea импорта → ранний return для полей ввода + защита от e.repeat.
4. Боссы были HP-стенами вне фокуса автооружия → приоритет босса у молнии/снарядов, пробитие заслона, HP 1500·t^1.75 → 1300·t^1.5, XP со скейлом.
5. «Спешка» не действовала на ауру/клинки → действует.
6. «Осколки боли» с фиксированными 18 урона → скейл от HP жертвы.
7. Минорные: грязный grid после рестарта, сгорание очереди уровней при пустом пуле, застрявший #lowhp, перекрывающиеся таймеры вспышки, слияние XP в сердца, санитизация импортируемого/загружаемого JSON (NaN-золото, up:null), телепорт орды при ресайзе, «Жадность» не множила бонусное золото.

### Цикл 6 — финальная верификация
Регрессия всех фиксов зелёная: битый импорт отклоняется/санитизируется, рестарт/выход чистые, пробел в textarea работает, босс убиваем в толпе (бухгалтерия урона сошлась: 147×18 + 1300 ≈ 5395), молния рендерится (проверено пиксельным замером канваса — 758/765 яркости). Исправлен последний косметический дефект — швы между тайлами земли (округление трансляции камеры). Итог: 257 врагов — 16.57ms avg / 19.4ms p95, **0 сообщений в консоли за все сессии**. Существенных улучшений на этом цикле не найдено — остановка по критерию ТЗ.

## Обновление 2: щит гнили, звук, расширение контента

### Кольцо гнили — щит от снарядов
По запросу: аура растворяет вражеские снаряды с градацией мощности. Слабые снаряды (урон ≤ 4+2·уровень) гасятся мгновенно на границе кольца; мощные (снаряды боссов высоких тиров) разъедаются постепенно — (10+7·уровень) урона/сек, пока снаряд внутри, и либо растворяются, либо долетают ослабленными. Проверено: 12 плевунов вокруг — 145 снарядов растворено, 0 урона; босс 3-го тира — снаряды частично пробивают. Новая синергия «Гнилой бастион» (aura≥3 + броня≥2): растворение ×2, погасшие снаряды взрываются во вред врагам. Счётчик «Снарядов растворено» на экране смерти.

### Звук
- **Баг «слышен звук при нулевой громкости» устранён радикально**: при master=0 весь AudioContext физически suspend'ится (проверено: state='suspended'), при music=0 секвенсор перестаёт планировать ноты. Аудио также глушится при сворачивании вкладки.
- Добавлен **DynamicsCompressor** на мастер-шине — наслоение SFX больше не клиппится.
- Все SFX пересведены и стали слоёными: выстрел = тон+щелчок шума, ранение = пила+саб+шум, смерть крупного врага звучит ниже и тяжелее, рык босса = два расстроенных саза+саб-синус, смерть босса и игрока — с минорными «хвостами»-колоколами, подбор кристаллов — с октавной искрой поверх растущего комбо-питча. Новые звуки: растворение снаряда, свист серпа, бросок/разрыв горшка.
- Музыка богаче: бас дублируется квинтой на пике, арпеджио меняет направление по тактам, сбивка снейра в конце круга, hi-hat'ы с «человеческой» вариацией громкости, лид-мотив со своей фразой на каждый аккорд.

### Локации (выбор в главном меню, открываются за рекорд выживания)
| Локация | Условие | Особенности |
|---|---|---|
| **Погост** | открыт сразу | классика |
| **Кровавая топь** | рекорд ≥ 4:00 | кровавые лужи замедляют и игрока, и врагов на ~18% (тактика позиционирования); слизни с первой минуты и в избытке; +15% золота |
| **Пепелище** | рекорд ≥ 7:00 | огненные разломы: телеграф 1.25с → взрыв, бьющий и игрока, и врагов; элита каждые 60с вместо 78; летающие угли; +25% золота |
Каждая локация — свой тайл земли, набор декора и палитра. Выбор хранится в мета-сейве.

### Новые орудия (всего 6) + лимит билда
- **Серп охотника** — бумеранг: летит сквозь толпу с торможением, возвращается к охотнику и рубит тех же врагов второй раз. До 3 серпов на 5-м уровне.
- **Чумной горшок** — мортира: лоб через головы в гущу врагов, разрыв + кислотная лужа (DoT + замедление) на 3 секунды. До 3 горшков.
- **Лимит 4 оружия за забег**: пятое новое оружие не предлагается — выбор билда стал решением.

### Новые прокачки и синергии (пул: 6 оружий + 12 пассивок + 7 синергий = 25)
- Пассивки: «Тёмный договор» (+25% урона, −15 max HP), «Голод пустоты» (+25% опыта), «Алхимия» (сердца лечат сильнее и чаще падают).
- Синергии: «Кровавая жатва» (серп≥3 + жажда крови≥2: порезанные истекают кровью 3с), «Чумной ветер» (горшок≥3 + спешка: +1 горшок, лужи шире), «Гнилой бастион» (см. выше).

### QA цикла расширения
Headless-прогон: консоль — 0 сообщений; щит ауры (145 блоков / частичное пробитие боссом); серп 961 урона + кровотечение тикает; горшок 1346 урона, лужи рисуются; Топь (замедление в лужах, слизни, ×1.15 золота) и Пепелище (разломы нанесли 510 урона врагам, угли) — скриншоты qa7_*.jpeg; лимит оружия (при 4 взятых новые не выпадают за 60 роллов); рестарт чистит зоны/горшки/разломы; 258 врагов + лужи + разломы — 16.65ms/кадр (60 FPS).

## Известные ограничения
- Управление только клавиатурное (тач-управления нет — десктопный жанр).
- Музыка стартует после первого ввода (ограничение autoplay-политики браузеров).
- При выходе в меню из паузы золото за забег начисляется молча (решение: игрок не должен терять прогресс, но отдельного экрана для «сдался» нет).
- Тестировалось в Chromium headless; Safari/Firefox не проверялись (используются только стандартные API).
